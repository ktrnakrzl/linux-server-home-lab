# 🔧 Linux Server Home Lab

## 📌 Overview
This project demonstrates the deployment and configuration of a Linux server using Ubuntu Server in a virtualized environment. It includes web server setup, remote access configuration, basic security implementation, and real-world troubleshooting.

---

## 🧰 Technologies Used
- Ubuntu Server (Linux)
- VirtualBox
- Nginx Web Server
- SSH (Secure Shell)
- Fail2Ban
- Bash / Linux CLI

---

## ⚙️ Environment Setup

- **Platform:** VirtualBox
- **Operating System:** Ubuntu Server
- **Network:** Local network (private IP)
- **Access Method:** SSH from host machine
