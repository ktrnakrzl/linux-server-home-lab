# linux-server-home-lab
Linux server home lab project using Ubuntu Server, featuring Nginx web hosting, SSH configuration, and basic security with Fail2Ban.

🔧 Linux Server Home Lab
📌 Overview

Deployed and configured a Linux server using Ubuntu Server in a virtualized environment. This project demonstrates system administration, server setup, and security hardening.

⚙️ Setup Environment
OS: Ubuntu Server
Platform: VirtualBox
Network: Local network (SSH access via IP)
🌐 Web Server Setup (Nginx)
Installed Nginx
Configured web server
Hosted a test webpage
🔐 SSH Configuration
Enabled SSH access
Configured authentication settings
Tested remote login from host machine
🛡️ Security (Fail2Ban)
Installed and configured Fail2Ban
Simulated brute-force attacks using SSH
Monitored logs using journalctl
🧪 Troubleshooting & Issues Resolved
Fixed package installation errors
Diagnosed network connectivity issues
Resolved SSH access problems
Debugged Fail2Ban log detection issues
🧠 Key Learnings
Linux server management
Log analysis and troubleshooting
Network configuration and SSH
Basic system security practices
